# 📊 Sales Performance Intelligence Dashboard

An **End-to-End Business Intelligence project** that transforms raw transactional sales data into actionable insights using **SQL, data modeling, and Power BI visualization**.

This project demonstrates how organizations can leverage **data analytics and BI tools** to monitor sales performance, identify revenue drivers, and make data-driven business decisions.

---

# 🧾 Project Overview

Many organizations store large volumes of transactional data but struggle to convert it into strategic insights.

This project builds a **complete analytics pipeline**:

1. **Database Design (MySQL)** – Structured relational database for storing sales data
2.  **Data Modeling** – Fact and dimension tables using a **Star Schema**   
3. **Data Analysis (SQL)** – Queries used to extract business insights 
4. **Data Visualization (Power BI)** – Interactive dashboard showing key business KPIs

# 🧱 Data Warehouse Design (Star Schema)

To enable efficient analytics, the dataset follows a **Star Schema data model**, commonly used in **Business Intelligence systems**.

This model separates data into:

- **Fact Tables** → transactional metrics  
- **Dimension Tables** → descriptive attributes  

---

# ⭐ Star Schema Architecture

```
                Dim_Customers
                       │
                       │
Dim_Regions ─── Fact_Sales ─── Dim_Products
                       │
                       │
                Dim_Employees
                       │
                       │
                Dim_Categories
```

The **Sales table acts as the central fact table**, connecting all dimensions.

# 🧮 Key SQL Analysis Queries

### Total Revenue

```sql
SELECT SUM(total_amount) AS total_sales
FROM sales;
```

### Monthly Sales

```sql
SELECT YEAR(sale_date) AS year,
       MONTH(sale_date) AS month,
       SUM(total_amount) AS total_sales
FROM sales
GROUP BY YEAR(sale_date), MONTH(sale_date);
```

### Best Performing Product

```sql
SELECT products.product_name,
       SUM(sales.total_amount) AS revenue
FROM sales
JOIN products ON sales.product_id = products.product_id
GROUP BY products.product_name
ORDER BY revenue DESC
LIMIT 1;
```

### Revenue by Region

```sql
SELECT regions.region_name,
       SUM(sales.total_amount) AS revenue
FROM sales
JOIN customers ON sales.customer_id = customers.customer_id
JOIN regions ON customers.region_id = regions.region_id
GROUP BY regions.region_name;
```

# 📊 Power BI Measures (DAX)

### Total Revenue

```DAX
Total Revenue =
 SUM('business_intelligence_sales sales'[total_amount])
```

### Best Month

```DAX
Best Month = 
VAR MonthlySummary =
    SUMMARIZE(
        'business_intelligence_sales sales',
        'business_intelligence_sales sales'[YearMonth],
        "MonthlyRevenue", [Total Revenue]
    )

VAR TopMonth =
    TOPN(
        1,
        MonthlySummary,
        [MonthlyRevenue],
        DESC
    )

RETURN
MAXX(TopMonth, 'business_intelligence_sales sales'[YearMonth])
```

### Peak Performance

```DAX
Peak Performance Revenue Change = 
VAR MonthlyRevenue =
    SUMMARIZE(
        'Calendar',
        'Calendar'[YearMonth],
        "Revenue", [Total Revenue]
    )

VAR BestMonthTable =
    TOPN(1, MonthlyRevenue, [Revenue], DESC)

VAR BestRevenue =
    MAXX(BestMonthTable, [Revenue])

VAR BestMonth =
    MAXX(BestMonthTable, Calendar[YearMonth])

VAR PreviousRevenue =
    CALCULATE(
        [Total Revenue],
        FILTER(
            ALL('Calendar'),
            'Calendar'[YearMonth] =
            FORMAT(EDATE(DATEVALUE(BestMonth & "-01"), -1), "YYYY-MM")
        )
    )

RETURN
BestRevenue - PreviousRevenue
```

### Total Customers

```DAX
Total Customers =
DISTINCTCOUNT('business_intelligence_sales sales'[customer_id])
```

## 🚀 Strategic Recommendations

Based on the analysis of revenue, regional performance, and customer behavior, the following strategic actions are recommended to enhance overall business performance and profitability:

### 1️⃣ Expand High-Performing Categories
The **Electronics** category currently contributes the majority of total revenue, indicating strong customer demand and market opportunity. Recommended actions:  
- Allocate additional inventory and marketing budget to top-selling electronics products.  
- Introduce new product lines or variants within high-demand subcategories.  
- Monitor emerging consumer trends to ensure the product portfolio remains relevant and competitive.  

### 2️⃣ Strengthen Regional Marketing
Certain regions, particularly the **South region**, show significant revenue potential that is currently underutilized. Recommended actions:  
- Implement targeted marketing campaigns tailored to regional demographics and preferences.  
- Partner with local influencers or events to increase brand visibility and engagement.  
- Track campaign effectiveness using region-specific KPIs to optimize resource allocation.  

### 3️⃣ Improve Customer Retention
Repeat purchases are a key driver of sustainable revenue growth. Recommended actions:  
- Launch **loyalty programs** offering points, discounts, or exclusive access for returning customers.  
- Personalize communications based on purchase history and preferences to enhance engagement.  
- Introduce subscription-based services or bundled offers that encourage recurring orders.  

### 4️⃣ Develop Upselling and Cross-Selling Strategies
Upselling and cross-selling can significantly increase **average order value**. Recommended approaches:  
- Promote complementary products or accessories alongside premium purchases at checkout.  
- Train sales representatives to identify opportunities for upselling based on customer behavior.  
- Use data-driven recommendations on the website and marketing emails to encourage add-on purchases.  

### 5️⃣ Enhance Sales Team Performance
Sales team efficiency directly impacts revenue generation. Recommended actions:  
- Set clear, measurable **KPIs** for sales representatives tied to revenue, customer satisfaction, and conversion rates.  
- Implement incentive programs to reward top performers and motivate consistent achievement.  
- Provide ongoing training on product knowledge, negotiation skills, and customer relationship management.
